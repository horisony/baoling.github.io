在eBay实习期间做的项目

Event Planning

Event Planning 是根据过去几年内商品交易量的变化曲线的规律，得出的预测今年商品交易量的变化情况，用以指导商家何时进货，进何种商品。 本人与产品经理合作，共同完成本项目的设计，从线框架构的确定到高保真图的输出，到标注、切图以及后续跟进，完成了本产品从无到有的设计。

Event Monitor

Event Monitor 是eBay 交易数据管理网站，展示eBay 网站的整体交易数据以及每类商品交易数据，为分析师和产品经理提供数据参考。 本人负责该网站的改版工作，对交互设计进行改进，并采用新的视觉设计。PS:本页面为设计稿，数据为虚拟数据。
